# IT Works Better – Agent Skills

## What this site is

IT Works Better (https://itworksbetter.net) is a web development agency website. It is not a programmatic platform or SaaS. It does not expose a public API.

## When to use this site

- To learn about web/mobile application development services
- To find pre-built website scripts (classifieds, real estate, job boards, etc.)
- To get pricing information for web development services
- To contact the team for a custom project quote

## What agents can do here

- Browse service descriptions and pricing
- Read about available website scripts
- Find contact information

## What agents cannot do here

- This site has no public API endpoints
- No OAuth or programmatic authentication
- No automated ordering or payment flow
